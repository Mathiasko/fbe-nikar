<task type="auto">
  <name>Task 2: Create GalleryCard and GalleryGrid components</name>
  <files>src/components/GalleryCard.astro, src/components/GalleryGrid.astro, src/styles/global.css</files>
  <action>
1. Create src/components/GalleryCard.astro:
   - Props: imageSrc (ImageMetadata), title (string), subtitle (string), category (string), categoryColor (string)
   - Use Astro's Image component: `import { Image } from 'astro:assets';`
   - Structure:
     ```
     <article class="gallery-card">
       <Image src={imageSrc} alt={title} class="gallery-card__image" />
       <div class="gallery-card__content">
         <h3 class="gallery-card__title">{title}</h3>
         <p class="gallery-card__subtitle">{subtitle}</p>
       </div>
       <span class="gallery-card__category" style={`background-color: ${categoryColor}`}>{category}</span>
     </article>
     ```
   - Category colors from current site: brown (#6B4423), olive (#6B7B00), sage (#7B8B4B)

2. Create src/components/GalleryGrid.astro:
   - Wrapper component that renders children in CSS grid
   - Props: none (uses slot)
   - Just a semantic wrapper with grid class

3. Add CSS to src/styles/global.css:
   - .gallery-grid: CSS Grid, responsive (1 col mobile, 2 col tablet, 3-4 col desktop)
   - .gallery-card: Card styling matching current site aesthetic
   - .gallery-card__image: object-fit: cover, aspect-ratio for consistent sizing
   - .gallery-card__category: Colored band at bottom of card
   - Use gap: 1.5rem for spacing

CSS grid approach (no JS):
```css
.gallery-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
  gap: 1.5rem;
}
```
